5  /* Rockwell International
6  * The Rockwell chipset is used on a number of gps receivers. It outputs
7  * some proprietary sentences with the PRWI prefix and accepts input from
8  * some special sentences similar to the approach used by Magellan. It can
9  * also be switched to a separate binary mode using a proprietary sentence.
10  * The input sentence most used to initialize the unit is $PRWIINIT and one
11  * output sentence is $PRWIRID
12  *
13  * $PRWIRID,12,01.83,12/15/97,0003,*42
14  *
15  * where:
16  * $PRWIRID
17  * 12 12 channel unit
18  * 01.83 software version
19  * 12/15/97 software date
20  * 0003 software options (HEX value)
21  * Bit 0 minimize ROM usage
22  * Bit 1 minimize RAM usage
23  * *42 checksum
24  *
25  * An input sentence that will define which NMEA sentences are to be output
26  * from the Rockwell unit is:
27  *
28  * $PRWIILOG,GGA,A,T,1,0
29  *
30  * where:
31  * $PRWIILOG
32  * GGA type of sentence
33  * A A=activate, V=deactivate
34  * T cyclic
35  * 1 every 1 second
36  * 0 ??
37  *
38  * The initialization sentence which can be input to speed up acquisition looks like:
39  *
40  * $PRWIINIT,V,,,4308.750,N,07159.791,W,100.0,0.0,M,0.0,T,175244,230503*77
41  *
42  * where:
43  * $PRWIINIT INIT = initialization
44  * V V = reset, A = no reset
45  * ,, Reserved for future use
46  * 4308.750 Latitude
47  * N N = North, S = South
48  * 07159.791 Longitude
49  * W W = West, E = East
50  * 100.0 Altitude in meters
51  * 0.0 Speed
52  * M M = m/s, N = knots, K = km/hr
53  * 0.0 Heading
54  * T T = True, M = Magnetic
55  * 175244 UTC time (hour, min, sec)
56  * 230503 UTC date (day, month, year)
57  * *77 Checksum
58  *
59  * Note: Commas may be used to signify using existing data. If units
60  * are supplied then the data must be present. Speed and direction must
61  * be supplied together. Lat/Lon must be supplied together. UTC time and
62  * date must be supplied together. If heading is magnetic then lat/lon
63  * needs to be supplied along with UTC time and date.
64  *
65  * The sentences available for the Rockwell Jupiter chipset are:
66  * GGA, GSA, GSV, VTG, RMC and some proprietary sentences.
67  */
68  }
69 }
