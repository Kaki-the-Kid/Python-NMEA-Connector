152  /****************************************************************************
153  * PSLIB
154  * Proprietary sentences are used to control a Starlink differential
155  * beacon receiver. (Garmin's DBR is Starlink compatible as are many
156  * others.) When the GPS receiver is set to change the DBR frequency
157  * or b/s rate, the "J" sentence is replaced (just once) by (for example):
158  *
159  * $PSLIB,320.0,200*59 to set the DBR to 320 KHz, 200 b/s.
160  *
161  * $PSLIB,,,J*22 Status request
162  * $PSLIB,,,K*23 configuration request
163  *
164  * These two sentences are normally sent together in each group of
165  * sentences from the GPS. The three fields are: Frequency, bit Rate,
166  * Request Type. The value in the third field may be:
167  * J = status request,
168  * K = configuration request,
169  * or null (blank) = tuning message.
170  *
171  * The correct values for frequency range from 283.5-325.0 KHz while
172  * the bit rate can be set to 0, 25, 50, 100 or 200 bps.

196  /************************************************************************
197  * RTE - Routes
198  * RTE is sent to indicate the names of the waypoints used in an active route.
199  * There are two types of RTE sentences. This route sentence can list all of
200  * the waypoints in the entire route or it can list only those still ahead.
201  * Because an NMEA sentence is limited to 80 characters there may need to be
202  * multiple sentences to identify all of the waypoints. The data about the
203  * waypoints themselves will be sent in subsequent WPL sentences which will
204  * be sent in future cycles of the NMEA data.
205  *
206  * 1 2 3 4 5 x n
207  * | | | | | | |
208  * $--RTE,x.x,x.x,a,c--c,c--c, ..... c--c*hh<CR><LF>
209  *
210  * Field Number:
211  *
212  * 1. Total number of RE sentences being transmitted
213  * 2. Sentence Number
214  * 3. Sentence mode
215  * c = complete route, all waypoints
216  * w = working route, the waypoint you just left, the waypoint
217  * you’re heading to, then all the rest
218  * 4. Route ID
219  * 5. Waypoint ID
220  * x. Additiobal waypint IDs
221  * More waypoints follow. Last field is a checksum as usual.
222  *
223  * The Garmin 65 and possibly other units report a $GPR00 in the same format.
224  *
225  * Example: $GPRTE,1,1,c,0*07
